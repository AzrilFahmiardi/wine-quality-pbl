# Setup - Script Pembahasan

## 📋 Tujuan Section
Setup adalah tahap persiapan awal untuk mengimport library dan dependencies yang diperlukan untuk analisis data wine quality.

## 📚 Library yang Digunakan

### 1. **Pandas (pd)**
- **Fungsi**: Manipulasi dan analisis data
- **Kegunaan dalam project**: 
  - Loading dataset dari CSV
  - Data cleaning dan preprocessing
  - Analisis deskriptif statistik
  - Handling missing values

### 2. **NumPy (np)**
- **Fungsi**: Operasi numerik dan array manipulation
- **Kegunaan dalam project**:
  - Kalkulasi matematis pada fitur
  - Array operations untuk data processing
  - Statistical calculations

### 3. **Matplotlib (plt)**
- **Fungsi**: Visualisasi data 2D
- **Kegunaan dalam project**:
  - Membuat histograms untuk distribusi
  - Line plots untuk time series (jika ada)
  - Subplots untuk perbandingan multiple features

### 4. **Seaborn (sns)**
- **Fungsi**: Visualisasi statistik yang lebih cantik berbasis matplotlib
- **Kegunaan dalam project**:
  - Heatmap untuk correlation analysis
  - Distribution plots yang lebih informatif
  - Pair plots untuk exploratory analysis

## 🎯 Rangkuman
Setup section mengimport 4 library utama yang akan digunakan sepanjang EDA dan modeling process untuk wine quality dataset.
